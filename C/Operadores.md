### Operadores aritméticos:

- ***Usados para operaciones aritméticas:***

| Operador | Descripcion    |
| -------- | -------------- |
| +        | Suma           |
| -        | Resta          |
| *        | Multiplicacion |
| /        | Division       |
| %        | Modulo         |
| ++       | Incremento     |
| --       | Decremento     |

### Operadores de comparación

Se usan para comparar valores con otros valores o variables

| Operador | Descripcion       |
| -------- | ----------------- |
| ``!=``   | Diferente a       |
| <=       | Menor o igual que |
| >=       | Mayor o igual que |
| ==       | Igual que         |
| <        | Menor             |
| >        | Mayor             |
### Operadores lógicos:
