```sql
- `SELECT` - extrae datos de la base de datos
- `UPDATE` - sube datos a la base de datos
- `DELETE` - elimina datos de la base de datos
- `INSERT INTO` - inserta nuevos datos en la base de datos
- `CREATE DATABASE` - crea una nueva bases de datos
- `ALTER DATABASE` - modificas una base de datos
- `CREATE TABLE` - crea una nueva tabla
- `ALTER TABLE` - modifica una tabla
- `DROP TABLE` - elimina una tabla
- `CREATE INDEX` - crea un índice (clave de búsqueda)
- `DROP INDEX` - elimina un indice 
```
