Al igual que en otros lenguajes de programación tenemos operadores lógicos, en Oracle SQL tenemos 
- AND
- OR
- NOT

```SQL
SELECT *
FROM TABLA
WHERE CAMPO = 'Valor' OR CAMPO = 'Valor';
```

```sql
SELECT *
FROM TABLA
WHERE NOT CAMPO = 'Valor';
```

```sql
SELECT *
FROM TABLA
WHERE CAMPO = 'Valor' AND CAMPO2 = 'Valor';
```
